## PROJECT OVERVIEW
This is a modern frontend web application built using a fast and scalable technology stack.

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## LOCAL DEVELOPMENT SETUP

PREREQUISITES:-

- Node.js

- npm
(Recommended: install Node using nvm)

## SETUP STEPS

- Clone the repository
  git clone <YOUR_GIT_URL>

- Navigate to the project directory
  cd <YOUR_PROJECT_NAME>

- Install dependencies
  npm install

- Start the development server
  npm run dev
  The application will start with hot reloading enabled.

## EDITING THE PROJECT
You can edit the project using:-

Any local IDE (VS Code recommended)
GitHub (edit files directly)
GitHub Codespaces


## BUILD FOR PRODUCTION
npm run build

## PREVIEW PRODUCTION BUILD
npm run preview

## DEPLOYMENT
Deploy the production build using any hosting provider that supports static or Vite-based applications.
